# TRS-80 Model III emulator

This is a TypeScript-based TRS-80 Model III emulator.
[Try it in your browser](https://lkesteloot.github.io/trs80-emulator/).

To run a local build:

    % npm install
    % npm run build
    % npm start

Then visit http://127.0.0.1:8080/
